# 💳 Smart Banking System

A full-stack banking application that allows users to perform core banking operations like account creation, deposit, withdrawal, and fund transfer.

---

## 🚀 Tech Stack

### Backend
- Spring Boot
- Java
- REST APIs

### Frontend
- React.js
- HTML, CSS, JavaScript

### Database
- MySQL

### Tools
- Postman (API Testing)
- Git & GitHub

---

## ✨ Features

- 🆕 Create Bank Account  
- 💰 Deposit Money  
- 💸 Withdraw Money  
- 🔁 Transfer Funds  
- 📊 View Account Details
